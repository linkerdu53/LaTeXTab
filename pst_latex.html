Comment utiliser le générateur de tableaux LaTeX ?

I. Définissez la taille souhaitée de la table grâce au + autour du tableau ou l'onglet nouveau tableau.

II. Entrez les données voulues dans le tableau.

III. Ajustez l'alignement du texte, les styles de police, la couleur du texte... en utilisant les boutons de la barre d'outils.
Le formatage est appliqué à toutes les cellules sélectionnées.

IV. Vous pouvez visualiser le rendu sur un PDF et si cela vous convient vous pouvez copier le code LaTeX grâce au bouton.
